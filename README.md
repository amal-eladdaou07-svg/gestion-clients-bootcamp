## Description

Cette application web permet la gestion complète des clients d’une entreprise
à travers une interface simple, intuitive et fonctionnelle.

Elle propose les opérations principales CRUD (Create, Read, Update, Delete),
ainsi qu’un système d’authentification utilisateur et un module de statistiques
permettant de visualiser l’activité liée aux clients.

Ce projet a été développé dans le cadre d’une formation Bootcamp afin de
renforcer les compétences en développement web avec PHP et MySQL.

---

## Fonctionnalités

* Ajouter un client
* Modifier les informations d’un client
* Supprimer un client
* Rechercher un client
* Afficher les statistiques des clients
* Authentification utilisateur (login / logout)

## Installation

1️⃣ Cloner le projet

git clone https://github.com/amal-eladdaou07-svg/gestion-clients-bootcamp.git

2️⃣ Copier le dossier dans :

C:\xampp\htdocs

3️⃣ Démarrer Apache et MySQL dans XAMPP

4️⃣ Importer la base de données dans phpMyAdmin

5️⃣ Ouvrir dans le navigateur :

http://localhost/gestion-clients-bootcamp
