## Description
Cette application web permet de gérer les clients dans une entreprise.  
Elle a été développée dans le cadre d’une formation Bootcamp.

## Fonctionnalités
- Ajouter un client
- Modifier les informations d’un client
- Supprimer un client
- Rechercher un client
- Afficher les statistiques des clients
- Authentification (login / signup)

## Technologies utilisées
- PHP
- MySQL
- HTML / CSS
- JavaScript
- XAMPP

## Installation

1. Cloner le projet :
git clone https://github.com/amal-eladdaou07-svg/gestion-clients-bootcamp.git

2. Copier le dossier dans :
C:\xampp\htdocs

3. Démarrer Apache et MySQL dans XAMPP.

4. Importer la base de données dans phpMyAdmin.

5. Ouvrir dans le navigateur :
http://localhost/mon_appp
